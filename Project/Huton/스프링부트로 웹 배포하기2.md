# CI 환경 구축하기

스크립트를 개발자가 직접 실행시킴으로써 스프링부트 프로젝트를 배포할 수 있디만 불편함이 따른다. 

따라서 개발자가 VCS 시스템(코드 버전 관리)에 Push하면 자동으로 Test, Build가 수행되고 Build 결과를 운영 서버에 배포까지 자동으로 진행하는 CI 환경을 구축해보자.
