# 배열

> **같은 데이터 타입**을 가진 **연속된 메모리 공간**으로 이루어진 자료구조

- 변수는 하나인데 값은 여러개 -> 배열은 참조형 타입이다.



### 배열 만들기

1. 생성하기

```java
int[] array1 = new int[100];
```

- 100개의 정수값을 저장할 수 있는 정수형 배열
- 배열은 선언할 때 만들어진 크기가 변하지 않는다

2. 값 저장하기

```java
// array[index] = value;

array1[0] = 1;
array1[99] = 100;
```

3. 선언과 동시에 초기화

```java
int[] array2 = new int[]{1, 2, 3, 4, 5};
int[] array3 = {1, 2, 3, 4, 5};
```

4. 값을 꺼내어 사용하기

```java
int value = array2[2];
System.out.println(value); // 3
```



### 배열 사용하기

for 반복문, `length`

```java
// 1부터 100까지 정수가 담긴 배열 만들기
int[] iarray = new int[100];

for(int i = 0; i < iarray.length; i++) { 
    iarray[i] = i + 1;
}
```



### 2차원 배열

1. 선언하기

```java
int[][] array4 = new int[3][4]; // 3행 4열 2차원 배열 생성
```

2. 값 저장하기

```java
array4[0] = 10; // 오류! > 또 다른 1차원 배열을 가리키는 참조형 변수
array4[0][0] = 10;
```

3. 가변크기의 2차원 배열 생성하기

```java
int[][] array5 = new int[3][]; // 실제로 값을 담는 1차원 배열은 만들어 지지 않음

array5[0] = new int[1]; // 1차원 배열을 생성해줘야한다.
array5[1] = new int[2]; // 각각 길이가 다른 배열을 만들어 줄 수 있음
array5[2] = new int[3];
```

4. 선언과 동시에 초기화하는 방법

```java
int[][] array6 = {{1}, {2, 3}, {4, 5, 6}};
```



### for each

> 자바 1.5 버전부터 추가된 문법

```java
// for(타입 값을 받아줄 변수명: 출력하고 싶은 자료 구조)

int[] iarr = {10, 20, 30, 40, 50};

for(int value: iarr) {
    System.out.println(value);
}
```
