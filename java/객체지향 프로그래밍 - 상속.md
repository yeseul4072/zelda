## 상속

자바는 클래스 이름 뒤에 `extends` 키워드를 적고 부모 클래스 이름을 적게 되면 부모 클래스가 가지고 있는 것을 상속받을 수 있게 된다. 

```java
public class Car {
    public void run() {
        System.out.println("달려달려~");
    }
}

public class Bus extends Car {
    
}
```

```java
public class BusExam {
    public static void main(String[] args) {
        Bus bus = new Bus();
        bus.run(); // 달려달려~
    }
}
```

- Bus class는 아무런 코드를 가지지 않지만 run 메소드를 사용할 수 있다. 

```java
public class Bus extends Car {
    public void ppangppang() {
        System.out.println("빵빵"); // 메소드 확장
    }
}
```

- 부모가 가지고 있는 메소드 외에 추가로 메소드를 선언하는 것을 `확장`하였다고 표현한다. 



### 상속과 접근 제한자 protected

접근 제한자 종류

- public : 어떤 클래스든 접근할 수 있는 것을 의미
- protected : 같은 패키지인 경우 접근 가능, **다른 패키지인 경우 상속 받은 자식 클래스는 접근 가능**

- default(아무것도 쓰지 않은 경우) : 자기 자신과 같은 패키지에서만 접근할 수 있는 것을 의미
- private : 같은 클래스 내에서만 접근할 수 있는 것을 의미

```java
package javaStudy;

public class AccessObj {
    public int p = 3;
    protected int p2 = 4;
    int k = 2;
    private int i = 1;
    
}

public class AccessObjExam {
    public static void main(String[] args) {
        AccessObj obj = new AccessObj();
        
        System.out.println(obj.p); // public 접근 가능
        System.out.println(obj.p2); // protected 접근 가능
        System.out.println(obj.k); // default 접근 가능
        System.out.println(obj.i); // private 접근 불가
    }
} 
```

}
