## Java 문법

### 증가, 감소 연산자

변수의 값을 1 더하거나 뺄 때 사용한다(++, --)

연산자가 항의 앞에 있는지, 뒤에 있는지에 따라 결과가 달라진다.

- 항의 앞에 있으면, 해당 statement(`;`붙은 한 문장)가 끝나기 전에 값을 증가/감소 시킨다.
- 반대로 항의 뒤에 있으면, 해당 statement가 끝난 후 값을 증가/감소 시킨다.

```java
int score = 100;

System.out.println(++score); // 101
System.out.println(score++); // 101
System.out.println(score); // 102, 출력한 후에 증가됨!
```



### 논리 연산자

연산의 결과가 true/false로 반환된다.

- `&&` 논리 곲
  - 두 항이 모두 true일 때만 결과가 true
  - **앞의 항이 false이면 뒤 항의 결과를 평가하지 않는다.**
- `||` 논리 합
  - 두 항이 모두 false일 때만 결과가 false
  - **앞의 항이 true이면 뒤 항의 결과를 평가하지 않는다.** 
- `!` 부정



### 삼항 연산자

```java
조건식? 결과1 : 결과2;
```

- 조건식이 true이면 결과 1, false이면 결과2 실행한다. 
- 간단한 조건문 대신 사용할 수 있는 연산자



### 비트 연산자

- `~`: 비트의 반전

- `&`: 비트 단위 AND

  1 & 1만 1 반환하며, 그 이외는 다 0 반환

- `|`: 비트 단위 OR

  0 | 0만 0 반환하며, 그 이외는 다 1 반환

- `^`: 두개의 비트가 서로 다른 경우에 1 반환
- `<<`: 왼쪽으로 shift
- `>>`: 오른쪽으로 shift
